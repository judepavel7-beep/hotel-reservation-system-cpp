# Hotel Reservation System (C++)

## Description
This project is a simple console-based hotel reservation management system written in C++.

The program allows basic management of hotel operations such as:
- managing rooms
- managing clients
- making reservations

## Technologies Used
- C++
- File handling
- Basic data structures
